## 功能

- **59 条 TextMate tokenColors** 覆盖 Vue SFC / HTML / CSS / JS / TS / JSON / Markdown
- **32 条 Semantic tokenColors** 覆盖 TS/JS 语义高亮
- **Vue 三区块着色**：`<template>` `#F2858C` / `<script>` `#FFD700` / `<style>` `#21bd9e`
- **Vue 指令高亮**：`v-if` 品红 / `:` 绑定紫色 / `@` 事件金色 / `#` slot 蓝色
- **CSS 全配色**：选择器 / 属性 / 值 / 伪类 / 变量 / 函数 / 单位 独立颜色
- **JS/TS 增强**：关键字 / 模板字符串 / 箭头函数 / 装饰器 / 正则 / 对象 key
- **JSON 配色**：key 与 value 颜色区分
- **Markdown 支持**：标题 / 粗体 / 斜体 / 代码 / 链接
- **括号引导线**：安装主题自动开启彩色配对连线
- **Cursor 专属 UI**：聊天面板 / 命令中心 暗色适配

## 安装

- Trae / VS Code / Cursor 扩展面板 → Install from VSIX → 选择 `.vsix` 包
- 或通过扩展市场搜索 **Strange Theme**
